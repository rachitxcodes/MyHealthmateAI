# MyHealthmateAI 🏥

A smart healthcare monitoring system that integrates hardware sensors with software to track vital health parameters in real-time.

## 🚀 Features
- ❤️ Heart Rate & SpO2 monitoring (MAX3010x)
- 🌡️ Body Temperature tracking (DS18B20)
- 📟 OLED display for live readings
- 🔗 ESP32-based data collection
- 💻 Backend + Frontend integration

## 📂 Project Structure
```
backend/        → API & server logic  
frontend/       → UI / web interface  
esp32_code/     → Sensor & hardware code  
```

## ⚙️ Tech Stack
- ESP32, Arduino
- Python (Backend)
- Web Technologies (Frontend)

## ▶️ How to Run
1. Upload code to ESP32 from `esp32_code/`
2. Run backend server
3. Start frontend
